# Introducción a los Sistemas de Información

Bienvenido al repositorio de entregas prácticas. Este espacio contiene las resoluciones, códigos y documentos correspondientes a los laboratorios, talleres y el proyecto final de la asignatura.

El propósito de nuestras entregas es aplicar el enfoque de sistemas, utilizar herramientas tecnológicas y resolver casos de estudio para consolidar los conceptos teóricos de la gestión de datos y la infraestructura tecnológica.

## Tecnologías y Prerrequisitos

Para ejecutar los laboratorios de código y talleres de este repositorio, asegúrate de contar con el siguiente entorno configurado:

Git: Para clonar el repositorio y gestionar las ramas de tu equipo de proyecto.

1. Python 3.8+: Entorno de ejecución de los laboratorios de datos.

2. Librerías de Python requeridas:

   pip install pandas matplotlib jupyter

4. Herramientas de Diseño (Recomendadas para el proyecto): Figma, Balsamiq o Draw.io para mockups y diagramado de bases de datos.
